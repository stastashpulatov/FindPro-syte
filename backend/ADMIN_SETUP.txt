📦 ЧТО НУЖНО СДЕЛАТЬ:

1. Установить зависимости:
   cd backend
   pip install -r requirements.txt

2. Запустить backend:
   uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

3. Открыть админку:
   http://localhost:8000/admin
